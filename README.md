# Object-Following Robot Car

## Project Overview
[cite_start]This project was developed for the **CSE211s Introduction to Embedded Systems (Spring 2026)** course at Ain Shams University, Faculty of Engineering[cite: 2, 3, 4]. 

[cite_start]The system is mounted on a robot car and is designed to follow an object moving in a straight line[cite: 9]. [cite_start]By continuously measuring the distance between the sensor and the obstacle, the car can maintain a suitable following distance and automatically stop safely if the leading object stops[cite: 7, 9].

## Hardware Requirements
* [cite_start]**Microcontroller:** Nucleo Board [cite: 7]
* [cite_start]**Sensor:** Ultrasonic Sensor (e.g., HC-SR04) [cite: 7]
* [cite_start]**Actuation:** Robot Car Chassis with DC Motors [cite: 9]
